Audity - Streamline the Audit Process
Guided Alpha Web App

Start on macOS:
1. Unzip the folder.
2. Double click START_MAC.command.
3. If macOS blocks execution, right-click the file and choose Open.

Start on Windows:
1. Unzip the folder.
2. Double click START_WINDOWS.bat.

Start on Linux:
1. Unzip the folder.
2. Run: chmod +x start-linux.sh
3. Run: ./start-linux.sh

The app starts a local web server on http://127.0.0.1:8787 and opens Audity in your browser.
No external hosted backend is used.
